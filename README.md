# 📚 Sistema de Notas Escolares

Este é um projeto simples desenvolvido em Python com o objetivo de praticar lógica de programação, estruturas de repetição, validação de dados e manipulação de listas/dicionários.

O sistema permite cadastrar alunos, registrar suas notas e consultar o desempenho individual com base em critérios de aprovação.

---

## 🚀 Funcionalidades

- Cadastro de alunos
- Validação de nomes (não permite entrada vazia)
- Validação de notas (0 a 10)
- Associação de alunos e notas
- Consulta de desempenho do aluno
- Classificação automática:
  - 10 → Excelente
  - 7 a 9 → Aprovado
  - 5 a 6 → Recuperação
  - 0 a 4 → Reprovado

---

## 🧠 Conceitos praticados

- Estruturas de repetição (`while`)
- Condicionais (`if / elif / else`)
- Listas (ou dicionários em versões futuras)
- Entrada de dados com `input()`
- Validação de dados
- Organização de lógica de programa
